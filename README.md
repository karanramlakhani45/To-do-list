## Todo List App

A simple and responsive Todo List web application built with React.js to help users manage their daily tasks efficiently. It provides a clean and interactive interface for organizing work and improving productivity.

---

## Features

- Add new tasks easily  
- Delete tasks when no longer needed  
- Mark tasks as completed  
- Real-time updates without page reload  

---

## Tech Stack

- React.js  
- JavaScript (ES6)  
- CSS  

---

## Run Locally

```bash
npm install
npm start
```

---

## Description

This project demonstrates fundamental React concepts such as components, props, and state management using hooks (useState). It also showcases how to handle user input, update the UI dynamically, and manage application state in a simple React application.
